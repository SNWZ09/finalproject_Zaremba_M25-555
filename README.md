# finalproject_Zaremba_M25-555

Данный проект представляет собой платформу для отслеживания и симуляции торговли валютами.

Консольные команды:
    register - позволяет зарегистрироваться новому пользователю.
    login - позволяет пользователю "войти в аккаунт" для управления валютами.
    show-portfolio - показывает валютный портфель пользователя.
    buy - позволяет купить валюту.
    sell - позволяет продать валюту.
    get-rate - позволяет сравнить валютные курсы.
    exit - выход из платформы.
